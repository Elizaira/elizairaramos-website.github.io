Buttons

Button have different colors and styles used for users to recognized what are the actions when it is clicked. There are different types of buttons: primary, secondary, success, danger, warning, info, light, dark, default and link buttons. Some web designers use buttons based on the color of the website or color of the logo if they have.

Input

Input are use in making forms like login page, entering the users email, password and other information of the users of the website. The input can be differ base on the type you will choose it is either type="email" type="password" type="search" type="number" and others.

You can also add label before the input tag to become your guide in the input you insert in your website.

Table

Table can have multiple row and columns base on the table you want. table tag is the first step you need to code to make a table. It is consist of tr for table row, td for table data (columns) th for table heading.

You can also add style color or border in your table in css.
